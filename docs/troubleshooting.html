Common Issues

Installation Issues

‚ùå "Dependencies are missing" error

Problem: The app shows that required dependencies are not installed.

Solutions:

  • Click "Install Missing Dependencies" and wait for completion
  • Try manual installation using Terminal:
    brew install python3 ffmpeg sox ollama
    pip3 install -U openai-whisper
  • Restart your terminal and try again
  • Check if Homebrew is properly installed: brew --version

‚ùå "Whisper: command not found" error

Problem: Whisper executable cannot be found during transcription.

Solutions:

  • Install Whisper via pipx: pipx install openai-whisper
  • Add pipx path to your shell: export PATH="$HOME/.local/bin:$PATH"
  • Restart the app after installing Whisper
  • Check if Whisper is in PATH: which whisper

‚ùå "Ollama service not running" error

Problem: Ollama service is not available for summarization.

Solutions:

  • Start Ollama service: ollama serve
  • Pull the required model: ollama pull llama3
  • Check if Ollama is running: curl http://localhost:11434/api/tags
  • Restart Ollama if it's not responding

Permission Issues

‚ùå "Microphone access denied" error

Problem: The app cannot access your microphone.

Solutions:

  • Go to System Preferences > Security & Privacy > Privacy > Microphone
  • Check the box next to "Meeting Note Taker"
  • If the app isn't listed, launch it once and try again
  • Restart the app after granting permission

‚ùå "Screen recording permission required" error

Problem: The app needs screen recording permission for audio capture.

Solutions:

  • Go to System Preferences > Security & Privacy > Privacy > Screen Recording
  • Check the box next to "Meeting Note Taker"
  • Click "Open System Settings" in the app if available
  • Restart the app after granting permission

Performance Issues

üêå Slow transcription processing

Problem: Transcription takes a very long time to complete.

Solutions:

  • Use a smaller Whisper model (tiny or base instead of large)
  • Close other applications to free up CPU resources
  • Ensure you have sufficient RAM (8GB minimum)
  • Check if your Mac is running hot and needs cooling

üíæ High memory usage

Problem: The app uses too much memory during processing.

Solutions:

  • Close other applications during processing
  • Use smaller models (tiny Whisper model)
  • Restart the app between long recordings
  • Consider upgrading your Mac's RAM if possible

Audio Issues

üîá No audio being recorded

Problem: The app shows recording but no audio is captured.

Solutions:

  • Check your microphone is working in other applications
  • Verify microphone permissions are granted
  • Test with a different microphone if available
  • Check System Preferences > Sound > Input for microphone levels

üì¢ Poor audio quality

Problem: Recorded audio is distorted or unclear.

Solutions:

  • Speak closer to the microphone
  • Reduce background noise
  • Adjust microphone input levels in System Preferences
  • Use an external microphone for better quality

Advanced Troubleshooting

Debug Mode

Enable debug logging to get more detailed error information:

# Run the app with debug logging
DEBUG=1 /Applications/Meeting\ Note\ Taker.app/Contents/MacOS/Meeting\ Note\ Taker

Reset Application Data

If the app is behaving unexpectedly, try resetting its data:

# Remove app preferences and cache
rm -rf ~/Library/Preferences/com.meetingnotetaker.*
rm -rf ~/Library/Caches/com.meetingnotetaker.*

Manual Dependency Check

Verify all dependencies are working correctly:

# Check Python
python3 --version

# Check Whisper
whisper --help

# Check Ollama
ollama list

# Check FFmpeg
ffmpeg -version
