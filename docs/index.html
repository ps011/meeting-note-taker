Installation

1

Download the App

Download the latest DMG file from the releases page and install it on your Mac.

2

Install Dependencies

The app will automatically check and install required dependencies:

  • Homebrew - Package manager
  • Python 3 - Runtime for Whisper
  • FFmpeg - Audio processing
  • Sox - Sound processing
  • Ollama - Local LLM runtime
  • Whisper - AI transcription
3

Configure Settings

Set up your Obsidian vault path and choose your preferred Whisper and Llama models.

4

Start Recording

Grant screen recording permissions and start capturing your meetings!
