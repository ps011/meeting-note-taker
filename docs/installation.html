Installation Steps

1

Download the Application

Download the latest DMG file from our releases page:

https://github.com/prasheelsoni/meeting-note-taker/releases/latest

Look for the file named Meeting Note Taker-1.0.0-arm64.dmg (or similar version number).

2

Install the Application

  1. Double-click the downloaded DMG file
  2. Drag the "Meeting Note Taker" app to your Applications folder
  3. Eject the DMG file
  4. Launch the app from Applications or Spotlight
3

Grant Permissions

When you first launch the app, you'll need to grant several permissions:

  • Microphone Access: Required for recording audio
  • Screen Recording: Required for capturing system audio
Important: After granting permissions, you must quit and relaunch the app for changes to take effect.
4

Automatic Dependency Installation

The app will automatically check and install required dependencies:

Homebrew

Package manager for macOS

Python 3

Runtime environment for Whisper

FFmpeg

Audio/video processing

Sox

Sound processing utilities

Ollama

Local LLM runtime

Whisper

AI transcription model

Note: This process may take several minutes depending on your internet connection and system performance.
5

Configure Settings

Set up your preferences in the app:

  • Obsidian Vault Path: Choose where to save your meeting notes
  • Whisper Model: Select transcription accuracy vs speed (tiny/base/small/medium/large)
  • Llama Model: Choose summarization model (llama3 recommended)
6

Test the Installation

Verify everything is working:

  1. Click the "Record" button
  2. Speak into your microphone for a few seconds
  3. Click "Stop" and wait for processing
  4. Check that transcription and summary are generated

Manual Installation (Advanced)

If you prefer to install dependencies manually or encounter issues with automatic installation:

Install Homebrew

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Install Dependencies

brew install python3 ffmpeg sox ollama

Install Whisper

pip3 install -U openai-whisper

Start Ollama Service

ollama serve

Pull Llama Model

ollama pull llama3

Common Issues

Permission Denied Errors

Solution: Make sure you've granted microphone and screen recording permissions in System Preferences > Security & Privacy.

Dependencies Not Found

Solution: Try running the manual installation commands above, or restart your terminal and try again.

Whisper Installation Fails

Solution: Try installing Whisper via pipx instead: pipx install openai-whisper
